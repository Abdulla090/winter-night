# Premium Color Palettes (Curated for 2026)

Copy and paste these into your Tailwind config or CSS variables.

## 1. "Obsidian & Viola" (Dark Mode SaaS)
A deep, serious dark mode with electric purple accents.
```css
:root {
  --bg-main: 240 10% 4%;      /* #09090B */
  --bg-card: 240 10% 7%;      /* #101012 */
  --border: 240 6% 15%;       /* #242428 */
  --text-main: 0 0% 100%;     /* #FFFFFF */
  --text-muted: 240 5% 65%;   /* #A1A1AA */
  --primary: 260 100% 65%;    /* Electric Purple */
  --primary-glow: 260 100% 65% / 0.2;
}
```

## 2. "Polar & Graphite" (Clean Light Mode)
Hyper-clean, medical grade white interface with sharp text.
```css
:root {
  --bg-main: 0 0% 100%;       /* #FFFFFF */
  --bg-subtle: 210 40% 98%;   /* #F8FAFC */
  --text-main: 222 47% 11%;   /* #0F172A */
  --text-muted: 215 16% 47%;  /* #64748B */
  --primary: 221 83% 53%;     /* #3B82F6 (Vibrant Blue) */
  --accent: 14 100% 63%;      /* #FF5A5F (Coral - Airbnb style) */
}
```

## 3. "Midnight Oil" (Developer Tool)
High contrast, hacker aesthetic but polished.
```css
:root {
  --bg-main: 0 0% 0%;         /* True Black */
  --text-main: 0 0% 90%;      /* Off White */
  --primary: 150 100% 50%;    /* Matrix/Neon Green */
  --secondary: 280 100% 50%;  /* Neon Purple */
  --border: 0 0% 20%;
}
```

## 4. "Glass & Clay" (Modern Consumer App)
Soft shadows, pastel tones, high blur.
```css
:root {
  --bg-main: 30 50% 98%;      /* Warm White */
  --card-bg: 0 0% 100% / 0.7; /* Translucent White */
  --text-main: 20 10% 20%;    /* Warm Grey */
  --primary: 10 80% 60%;      /* Terracotta */
  --shadow-color: 20 20% 80%;
}
```
